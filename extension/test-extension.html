🔔 Notifications Test

Verify that notifications are working:

Expected Notifications:
  • Media handoff: "Continue watching [title]?" with Continue/Dismiss buttons
  • Clipboard sync: "Clipboard Synced" with preview of copied text
How to verify:
  1. Open extension popup and ensure "Notifications" toggle is ON
  2. Grant notification permission if prompted
  3. Perform media handoff or clipboard sync test
  4. Look for Chrome notification in top-right corner

🔍 Debug Console

Check browser console for extension logs:

How to check:
  1. Right-click on this page → Inspect
  2. Go to Console tab
  3. Look for messages starting with "FlowLink"
  4. You should see: "FlowLink media detection loaded" and "FlowLink clipboard monitoring loaded"
Background Script Console:
  1. Go to chrome://extensions/
  2. Find FlowLink extension
  3. Click "Inspect views: background page"
  4. Check Console for connection status and message logs
