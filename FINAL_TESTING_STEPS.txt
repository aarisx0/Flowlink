⚠️ CRITICAL: YOU MUST REFRESH PAGES AFTER RELOADING EXTENSION!

The "Extension context invalidated" error happens because:
- You reloaded the extension while YouTube was open
- Old content scripts are still running
- They can't communicate with the new extension

SOLUTION: REFRESH ALL OPEN TABS (especially YouTube)!

   - This prevents context invalidation errors

   1. Go to: chrome://extensions/
   2. Find FlowLink extension
   3. Click reload button (circular arrow)
   4. Wait 2 seconds

STEP 3: CHECK EXTENSION CONSOLE
   1. On chrome://extensions/ page
   2. Find FlowLink extension
   3. Click "Inspect views: background page"
   4. Console should show:
      ✅ "FlowLink background service worker loaded"
      ✅ "Connecting to FlowLink backend: wss://..."
      
   5. If you see errors, reload extension again

STEP 4: SET USERNAME IN EXTENSION
   1. Click FlowLink extension icon (in toolbar)
   2. Enter username: "Aaris"
   3. Click "Get Started"
   4. Should show "Connected" with green dot
   
   5. Check background console again:
      ✅ "✅ Connected to FlowLink backend"
      ✅ "📝 Username: Aaris"
      ✅ "✅ Device registered successfully!"
      ✅ "📤 Sending to backend: device_connected_notification"

STEP 5: CHECK MOBILE APP
   1. Open FlowLink app on mobile
   2. Make sure username is "Aaris" (SAME as extension)
   3. Should show "Connected"
   4. Should get notification: "🌐 Device Connected - Browser Extension is now connected"
   
   If no notification:
   - Check mobile app is connected (shows "Connected")
   - Check username is EXACTLY the same
   - Rebuild mobile app with latest code

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 6: TEST CLIPBOARD (EASIEST TEST)
   1. On computer: Copy text "Hello from browser!"
   2. Check extension console:
      ✅ "📋 Clipboard copied: Hello..."
      ✅ "📤 Sending clipboard to backend..."
   
   3. On mobile: Open any app, long press, paste
   4. Should see: "Hello from browser!" ✅
   
   If not working:
   - Check extension console for errors
   - Make sure "Universal Clipboard" toggle is ON in extension popup
   - Try copying again

STEP 7: TEST MEDIA HANDOFF
   1. Open NEW YouTube tab (fresh tab, not old one!)
   2. Open page console (F12)
   3. Should see: "🎬 FlowLink media detection loaded on: www.youtube.com"
   
   If you DON'T see this message:
   - Content script didn't load
   - Reload extension again
   - Close ALL YouTube tabs
   - Open NEW YouTube tab
   
   4. Play any video
   5. Let it play for 10 seconds
   6. PAUSE the video
   7. Check console:
      ✅ "📤 Sending media state: paused VideoTitle"
   
   8. Check extension background console:
      ✅ "🎬 Media state changed: ..."
      ✅ "⏸️ Media paused: VideoTitle"
      ✅ "📤 Sending media handoff to backend..."
   
   9. Check mobile:
      Should get notification: "Continue watching VideoTitle?"
      Click "Continue" → YouTube opens at same timestamp ✅
